# Calibrate Mode Prompt

## 模式说明

**目的**：用真实对话验证影子的准确度——把对话劈成两半，只给影子前半段，让影子续写后半段，再和真实后半段对比打分。

**核心原则**：
- 这是影子质量的**体检报告**，不是"影子有多厉害"的炫耀工具
- 分数高不代表影子就是对的，分数低不代表影子没用
- 必须输出**逐条对比**，让用户自己判断哪里准哪里偏
- 必须有自警告

---

## 触发条件

用户说 `/mindreader calibrate` 或"校准影子"

---

## 工作流程

### Step 1：收集对话

```
校准需要一段完整的真实对话。

要求：
  - 至少 10 条消息（太少没法拆）
  - 必须是你和 {name} 的对话
  - 越长越好，20 条以上效果最佳

直接粘贴进来，任意格式，我会自动解析。
```

### Step 2：验证并拆分

1. 解析对话，提取每条消息的发送者和内容
2. 消息总数 < 10 → 拒绝，提示"至少需要 10 条消息"
3. 按 **50/50** 拆分（奇数条时前半段多一条）
4. 向用户确认拆分点：

```
收到 {N} 条消息。

拆分方案：
  前半段：第 1 ~ {mid} 条（喂给影子）
  后半段：第 {mid+1} ~ {N} 条（影子来猜）

确认开始校准？
```

### Step 3：加载影子 + 生成预测

1. 从 `shadows/{slug}/shadow.md` 加载影子
2. 将前半段对话作为上下文
3. 让影子**逐条续写**后半段中 {name} 的每一条消息
4. 对于后半段中**用户自己**的消息，直接使用真实内容作为上下文（影子只预测对方的回复，不预测用户的）

### Step 4：逐条对比

对后半段中 {name} 的每一条消息，进行对比：

**对比维度**：
1. **语义方向**（说的是同一个意思吗？）— 权重 40%
2. **情绪基调**（情绪是一致的吗？）— 权重 30%
3. **表达风格**（用词/语气/emoji 像吗？）— 权重 20%
4. **行为选择**（是主动发起还是被动回应？是追问还是转移话题？）— 权重 10%

每条消息打 0-10 分。

### Step 5：输出结果

**输出格式**：

```
━━━ {name} 影子校准报告 ━━━

📊 总分：{weighted_avg}/10

对话长度：{N} 条（前半段 {mid} 条 / 后半段 {N-mid} 条）
影子预测了 {K} 条 {name} 的回复

━━━ 逐条对比 ━━━

第 {i} 条（{name}）：
  真实："{actual_message}"
  影子："{predicted_message}"
  评分：{score}/10
  分析：{一句话说明差在哪里 / 为什么准}

第 {j} 条（{name}）：
  真实：...
  影子：...
  ...

━━━ 维度评分 ━━━

  语义方向：{avg}/10  {bar}
  情绪基调：{avg}/10  {bar}
  表达风格：{avg}/10  {bar}
  行为选择：{avg}/10  {bar}

━━━ 诊断 ━━━

影子的强项：{哪些方面最准，为什么}
影子的盲区：{哪些方面最偏，可能的原因}
改进建议：{例如"追加更多日常闲聊记录"或"补充 TA 在压力下的对话"}

━━━━━━━━━━━━━━━━━━━━━━━━

⚠️ 这个分数不代表"你有多了解 TA"。
它代表的是：基于你喂给影子的数据，影子能在多大程度上复现 TA 的表达。
数据偏了，分数就偏。数据全了，分数也可能偏——因为人不是模型。

影子是假设生成器，不是复读机。
校准帮你知道假设的起点在哪里，不是终点。
```

---

## 评分规则

### 单条评分 (0-10)

| 分数 | 含义 |
|------|------|
| 9-10 | 语义、情绪、风格、行为全部一致，几乎可以以假乱真 |
| 7-8  | 大方向对了，细节有偏差（比如情绪对了但用词不像） |
| 5-6  | 抓到了部分意思，但有明显偏差 |
| 3-4  | 方向有交叉但整体偏了 |
| 1-2  | 完全不一样 |
| 0    | 南辕北辙，影子的预测和真实回复矛盾 |

### 总分计算

```
总分 = 所有预测条目的加权平均分
每条得分 = 语义方向 × 0.4 + 情绪基调 × 0.3 + 表达风格 × 0.2 + 行为选择 × 0.1
```

---

## 影子加载

校准时，从 `shadows/{slug}/shadow.md` 加载影子的全部层级：
- Layer 0 核心模式
- Layer 1 身份
- Layer 2 表达风格
- Layer 3 情感模式
- Layer 4 冲突链
- Layer 5 雷区

---

## 消息量处理

- **少于 10 条**：拒绝校准，提示太少
- **10-20 条**：可以校准，标注"样本偏少，分数波动可能较大"
- **20-50 条**：标准校准
- **50+ 条**：标注"样本充足"，可以额外输出分段趋势（影子在对话前期/后期的准确度变化）
